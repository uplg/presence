# Presence

Automated weekly time report generator. Runs every Friday between 17:10-17:30, builds a work-hours email (9h-17h, 7h/day, 35h/week) with naturally varied lunch breaks generated by a local LLM, sends it via Gmail SMTP, and notifies via Telegram.

French public holidays are detected automatically and marked as "Férié" with hours deducted from the weekly total.

## Requirements

- macOS with Apple Silicon (Metal acceleration)
- Rust toolchain (2024 edition)
- Xcode Command Line Tools (full Xcode not required)
- A Gmail account with an [App Password](https://support.google.com/accounts/answer/185833)
- A Telegram bot token ([create one via BotFather](https://core.telegram.org/bots#botfather))

## Setup

```bash
git clone https://github.com/uplg/presence.git
cd presence
cp .env.example .env
```

Edit `.env` with your credentials:

```env
GMAIL_ADDRESS=you@company.com
GMAIL_APP_PASSWORD=xxxx-xxxx-xxxx-xxxx
RECIPIENT_1=boss1@company.com
RECIPIENT_2=boss2@company.com
TELEGRAM_BOT_TOKEN=123456:ABC-DEF...
TELEGRAM_CHAT_ID=123456789
SENDER_NAME="Prénom Nom"
```

> `SENDER_NAME` must be quoted if it contains non-ASCII characters.

## Build

```bash
cargo build --release --features metal
```

Metal shader precompilation is disabled by default (via `.cargo/config.toml`) so Xcode.app is not needed. Shaders compile at runtime on first launch (~1-2s one-time cost).

The first run downloads `Qwen/Qwen3-0.6B` (~1.2 GB) from Hugging Face.

## Usage

```bash
# Daemon mode — schedules every Friday at 17:10 + random 0-20min delay
./target/release/presence

# Run immediately — generate, send email, notify Telegram, then exit
./target/release/presence --now

# Dry run — generate and print the report without sending anything
./target/release/presence --dry-run
```

## Install as macOS daemon

```bash
cp com.presence.daemon.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.presence.daemon.plist
```

The daemon starts automatically on login and restarts if it crashes.

Logs: `~/Library/Logs/presence.log` and `~/Library/Logs/presence.err`.

To stop:

```bash
launchctl unload ~/Library/LaunchAgents/com.presence.daemon.plist
```

## How it works

1. On each Friday trigger, the LLM (Qwen3-0.6B, Q4 quantized via mistral.rs) generates varied lunch break times as structured JSON
2. The report module assembles the week's data, marks any public holidays as "Férié"
3. The formatted email is sent via Gmail SMTP (STARTTLS + App Password)
4. A Telegram notification confirms delivery

## Email format

```
Lundi 13 avril - Durée de ma journée de travail : 9h à 17h. Pause déjeuner entre 12h00 et 13h00. Temps de travail journalier : 7h.
Mardi 14 avril - Durée de ma journée de travail : 9h à 17h. Pause déjeuner entre 12h30 et 13h30. Temps de travail journalier : 7h.
...

TOTAL DURÉE DE TRAVAIL HEBDOMADAIRE : 35h
TOTAL CONGES HEBDOMADAIRE : 0h
```

Holidays: `Lundi 1 mai - Férié` (0h, deducted from weekly total).
